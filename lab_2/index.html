Gravity is a SAAS based front end developer framework that makes building websites easier

The lowdown

Gravity is SAAS based framework for making powerful, easily maintanable HTML5 websites. It helps you rapidly prototype your ideas and will generally make your life much easier

There are all kinds of useful features built in, including ready made typography presets, media queries, button styles, form styling, column sizing based on variables and much more. You will need to install SASS to run the framework but, to be honest, if you're not already using SASS you're really missing out.

Gravity uses Slim as the primary template engine.

How to download it?

Gravity is available from Github. Simple copy the files and you're good to go. In OSX, enter tge terminal and type the following:

git clone http://github.com/owainlewis/Gravity-Framework.git [project-name]

cd [project-name]

rake sass
